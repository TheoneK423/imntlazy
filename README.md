# I'm not lazy.

一个运行在 Windows 上的本地专注工具，用来在你专注时**限制可用窗口、屏蔽娱乐网站、定时安排工作/休息循环，并在人离开电脑时强制弹出全屏提醒**。

## 主要功能

### 1. 窗口白名单

进入专注模式后，程序会持续轮询当前可见窗口，并将**不在白名单中的窗口最小化**。  
你可以在“允许窗口”里选择哪些程序或窗口标题允许保留。

### 2. 网站屏蔽

专注开始时，程序会修改系统 `hosts` 文件，临时屏蔽你设置的域名。  
专注结束后会自动移除对应条目。

默认示例：

- `zhihu.com`
- `bilibili.com`
- `youtube.com`

### 3. 定时专注循环

程序支持总专注时长，以及每轮工作 / 休息时长设置。  
默认配置是：

- 总时长 **180 分钟**
- 工作 **45 分钟**
- 休息 **15 分钟**

如果总时长会落在某段休息时间里，程序会在这段休息开始时直接结束整轮专注。

### 4. 停止专注确认

为了防止随手退出专注，你可以设置一段**确认短语**。  
手动停止专注时，必须输入完全一致的字符串才能确认。

### 5. 人脸检测提醒

可选启用摄像头人脸检测。  
当专注模式开启且连续一段时间未检测到人脸时，程序会显示**全屏遮罩提醒**；检测到人脸后会自动解除。

提醒界面会：

- 全屏置顶显示
- 播放蜂鸣声（可关闭）
- 阻止键盘和鼠标输入，直到检测到人回来

## 使用方式

### 直接运行

打包后的程序本体是：

`dist\imntlazy.exe`

双击即可运行。程序会常驻托盘，关闭主窗口不会退出程序，只会隐藏到托盘。

### 开发运行

```bash
pip install PySide6 opencv-python pyinstaller
python run.py
```

### 打包

```bash
pyinstaller imntlazy.spec --noconfirm --distpath dist
```

## 设置保存位置

程序设置保存在：

`%APPDATA%\imntlazy\settings.json`

通常对应：

`C:\Users\你的用户名\AppData\Roaming\imntlazy\settings.json`

这里会保存：

- 白名单窗口
- 屏蔽网站列表
- 专注总时长 / 工作时长 / 休息时长
- 停止专注确认短语
- 人脸检测开关
- 蜂鸣提醒设置

## 系统文件变更

网站屏蔽功能会临时修改：

`C:\Windows\System32\drivers\etc\hosts`

程序会用下面两行标记包住自己写入的内容：

- `# imntlazy Block Begin`
- `# imntlazy Block End`

## 注意事项

- 这是一个 **Windows 专用** 工具。
- 程序需要管理员权限运行，才能修改 `hosts` 文件和执行部分系统级限制。
- 所有功能都在本地运行，不依赖在线服务。
